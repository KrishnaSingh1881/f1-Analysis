# From P14 to "Simply Lovely" — 2022 Belgian GP Analysis

This project is a data-driven statistical analysis of the 2022 Belgian Grand Prix at Spa, focusing on how Max Verstappen was able to win from P14 on the grid.

**Drivers analyzed:** VER, PER, SAI, RUS

**How to run:**

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Open `notebook.ipynb` in Jupyter (or your preferred notebook environment) and run the cells.
